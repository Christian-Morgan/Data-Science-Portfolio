# Documentation
This folder contains documentation and supporting materials for the e-commerce data pipeline project.

## Contents
`/data-factory`

Documentation for the Azure Data Factory pipeline, including:

- Linked service configurations (Azure SQL Database, Azure Databricks)
- Pipeline architecture and flow
- Trigger and scheduling setup
- Screenshots of pipeline components

`/sql`

Documentation for the SQL Server Change Data Capture (CDC) setup:

- CDC script overview and purpose
- Database configuration details
- Tables tracked for incremental ingestion
